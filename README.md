# Cyber-Life: Эволюционная симуляция ботов

Проект представляет собой визуальную симуляцию цифровой жизни, где боты с простыми нейросетями учатся выживать, получать энергию, размножаться и взаимодействовать друг с другом.  
Каждый бот имеет собственный "мозг" и способен мутировать при делении, создавая новые поколения с изменённым поведением.

---

## Возможности

- Эволюция с наследованием и мутацией нейросетей  
- Фотосинтез — получение энергии от окружения  
- Атака и обмен энергией между ботами  
- Размножение при достижении достаточного уровня энергии  
- Старение и смерть при исчерпании ресурсов  
- Простая нейросеть (9 входов, 10 скрытых нейронов, 6 выходов)

---

## Зависимости

Для сборки и запуска необходимы:

- [Raylib](https://www.raylib.com/) — библиотека для графической визуализации  
- Компилятор C (например GCC)

---

## Сборка и запуск
```
git clone https://github.com/CABALABA1212/cyber-life.git
cd cyber-life
make
./main || main.exe
```
## Описание работы

- При запуске открывается окно 1280x720  
- На поле случайным образом размещаются боты  
- Каждый бот имеет энергию, возраст и направление  
- Боты совершают действия: движение, фотосинтез, деление, атака, обмен  
- Если все боты погибают — автоматически создаётся новая популяция

---

## Основные структуры

- **Bot** — хранит состояние бота: позицию, энергию, цвет, возраст, нейросеть  
- **Brain** — простая нейросеть прямого распространения  
- **init()** — инициализация начальной популяции  
- **mutate_brain()** — мутация весов и смещений нейросети  
- **step()** — главный цикл симуляции, выполняющий обновление логики, взаимодействий и рендеринг

## Визуализация

Каждый бот отображается как цветной квадрат.  
Цвет определяет "род" или линию потомков.  
Со временем можно наблюдать эволюцию поведения и выживание наиболее приспособленных особей.
